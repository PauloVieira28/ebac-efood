EFOOD - Part 1
About the Project

This project is the first part of the EFOOD application, developed as a front-end exercise based on a Figma design provided during the course.

The main objective was to recreate the proposed interface using React, TypeScript, and Styled Components, applying the concepts learned throughout the course.

This project was developed independently, without direct assistance from the instructors. Whenever necessary, additional research was done using official documentation and other reliable online resources to better understand the concepts presented in the course and apply them to the project.

Note: This is only the first stage of the project. New features and improvements will be added in the next parts.

Technologies
React
TypeScript
Vite
Styled Components
React Router DOM
Getting Started
Prerequisites

Before running the project, make sure you have installed:

Node.js (version 18 or later recommended)
npm
Installation

Clone the repository:

git clone <repository-url>

Go to the project folder:

cd <project-folder>

Install the dependencies:

npm install

Start the development server:

npm run dev

Then open the URL shown in the terminal (usually http://localhost:5173).
